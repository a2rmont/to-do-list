# To-Do List ✅

Lista de tarefas interativa, com filtros e salvamento automático — feita com HTML, CSS e JavaScript puro (sem frameworks ou bibliotecas).

**[🔗 Ver projeto no ar](https://a2rmont.github.io/to-do-list/)**

---

### 📸 Preview

Adicione, conclua e filtre suas tarefas. Tudo é salvo automaticamente no navegador, então nada se perde ao recarregar a página.

### ⚙️ Funcionalidades

- ➕ Adicionar tarefas (pelo botão ou apertando Enter)
- ✅ Marcar tarefas como concluídas
- 🗑️ Remover tarefas
- 🔍 Filtrar por: todas, pendentes ou concluídas
- 💾 Persistência de dados com `localStorage` (as tarefas continuam salvas mesmo após recarregar a página)

### 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (Flexbox, animações com `@keyframes`)
- JavaScript puro (manipulação de DOM, arrays, eventos, `localStorage`)

### 🧠 O que pratiquei nesse projeto

- Delegação de eventos (event delegation) para lidar com elementos criados dinamicamente
- Persistência de dados no navegador com `localStorage` e `JSON.stringify`/`JSON.parse`
- Renderização condicional de listas com base em filtros de estado

### 🚀 Como rodar localmente

```bash
git clone https://github.com/a2rmont/to-do-list.git
cd to-do-list
```

Depois, basta abrir o arquivo `index.html` no navegador.
